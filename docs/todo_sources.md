
# Тодо-бичиг (Ойратская "ясная" письменность) - источники данных (v0.1)

Цель документа - собрать начальный список открытых источников с материалами на тодо‑бичиг (факсимиле, транслитерации, переводы, словари) для проекта. Для каждого источника указаны типы материалов, формат, юридические оговорки (не везде) и полезные примечания для последующей обработки.

> Файл обновляется по мере появления новых источников/ссылок/полезной информации связанной с тодо-бичиг.

---

## 1) Библиотеки, архивы и корпуса

### 1.1 Центр по развитию калмыцкого языка (ЦРКЯ) - раздел “Библиотека” (baylig.ru)
- **URL:** https://baylig.ru/library/
- **Что есть:** Категории *"Литература на Тодо бичиг"*, *"Рукописи на тодо бичиг"*, учебные и исторические материалы.
- **Формат:** PDF/сканы, иногда текст.


### 1.2 Научный архив Калмыцкого научного центра РАН (КалмНЦ РАН, Элиста)
- **URL:** https://kigiran.com/ - публикации об архивных фондах.
- **Что есть:** Буддийские и светские рукописи на тодо‑бичиг, фонды редких рукописей, описания единиц хранения, исследования с фрагментами и транслитерациями.
- **Формат:** Печатные каталоги, статьи (в формате PDF), иногда образцы сканов в публикациях.
- **Юридически:** Доступ/копирование по запросу архива, для публичного датасета нужна письменная договоренность.
- **Примечания:** Полезно для **эталонных примеров** нормализации/транслитерации, можно договариваться (наверное) об оцифровке и деперсонизации метаданных.


### 1.3 CNEAS, Университет Тохоку, Япония
- **URL (портал):** https://hkuri.cneas.tohoku.ac.jp/
- **Модули:** **K‑Search** (полнотекстовый поиск) и **K‑Dic** (многозадачные словари), включая **Todo**/**Oirat** разделы.
- **Что есть:** Полнотекстовые коллекции (Written Mongolian/Manchu/漢蒙), словари и справочники по **тодо‑бичиг/ойратскому**, есть **Integrated Dictionary of Written Oirat** (Kuribayashi, монография и онлайн-доступ).
- **Формат:** Веб‑интерфейс, иногда PDF; выгрузки не всегда доступны.
- **Юридически:** Академический доступ.
- **Примечания:** Отличный источник для **лексикографической валидации**, построения **глоссариев** и правил транслитерации.


### 1.4 Университет Кембриджа - материалы по "Tod Üsüg"
- **URL (пример):** https://www.repository.cam.ac.uk/items/109ddc08-1827-47bb-9174-03550bcbcb5c
- **Что есть:** Академические материалы о тодо‑бичиг, ссылки на источники.

## 2) Словари, грамматики и лингвистические ресурсы

### 2.1 Позднеев А. М. - "Калмыцко‑русский словарь" (1911) (публ. репринты)
- **URL:** https://altaica.ru/LIBRARY/mong/%D0%9F%D0%BE%D0%B7%D0%B4%D0%BD%D0%B5%D0%B5%D0%B2_%D0%9A%D0%B0%D0%BB%D0%BC%D1%8B%D1%86%D0%BA%D0%BE-%D1%80%D1%83%D1%81%D1%81%D0%BA%D0%B8%D0%B9%20%D1%81%D0%BB%D0%BE%D0%B2%D0%B0%D1%80%D1%8C%201911.pdf
- **Что есть:** Исторический калмыцко‑русский словарь. Ценен для нормализации/транслита (орфография и набор знаков может отличаться от современных практик, это необходимо уточнить).
- **Формат:** PDF (скан).
- **Юридически:** По идее в общественном достоянии.


### 2.2 Kuribayashi H. - "Integrated Dictionary of Written Oirat" (CNEAS Monograph + онлайн)
- **URL (монография PDF):** https://hkuri.cneas.tohoku.ac.jp/articles/roncho/a72sosho61.pdf
- **URL (портал словарей, раздел Todo/Oirat):** https://hkuri.cneas.tohoku.ac.jp/project1/kdic/list?groupId=32
- **Что есть:** Интегрированный словарь письменного ойратского языка, справочные материалы, пояснения.
- **Формат:** PDF и веб‑интерфейс.
- **Юридически:** Использовать с цитированием (по остальному сильно не углублялся).
- **Примечания:** База для **правил транслитерации** и **лексических нормализаций**.


### 2.3 Грамматические и обзорные работы (для правил)
- **Birtalan Á. — главы по Oirat (обзор):** https://theswissbay.ch/pdf/Books/Linguistics/Mega%20linguistics%20pack/Mongolic/Kalmyk%20Oirat%3B%20Oirat%20%28Birtalan%29.pdf
- **Омакaева Э. и др. — исследования корпуса рукописей и письменной традиции (подборка):** пример - https://univerlag.uni-goettingen.de/bitstream/handle/3/isbn-978-3-86395-554-0.8/schatz_multiethnic_societies_08_omakaeva.pdf
