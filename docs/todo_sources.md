
# Тодо-бичиг (Ойратская "ясная" письменность) - источники данных (v0.2)

Цель документа - собрать начальный список открытых источников с материалами на тодо‑бичиг (факсимиле, транслитерации, переводы, словари) для проекта. Для каждого источника указаны типы материалов, формат, юридические оговорки (не везде) и полезные примечания для последующей обработки.

> Файл обновляется по мере появления новых источников/ссылок/полезной информации связанной с тодо-бичиг.

---

## 0) Коротко про Unicode для тодо бичиг

- Письмо тодо‑бичиг кодируется в **блоке Mongolian U+1800–U+18AF** (единая модель для hudum Mongolian / Todo / Manchu / Sibe). В блоке есть собственные **Free Variation Selectors** (FVS1–FVS4) и **MVS U+180E (Mongolian Vowel Separator)**.
- **MVS U+180E** используется для разделения перед финальными **A/E** и выбора нужной формы букв; исторически применявшийся **NNBSP U+202F** сегодня **не рекомендован** для монгольской письменности; в современной практике рекомендуют **MVS** (см. UTN #57 и TR/обсуждения имплементаций).
- Для корректного рендеринга требуются: шрифт (напр., **Noto Sans Mongolian**, **Mongolian Baiti**) + движок шейпинга (HarfBuzz/DirectWrite/CoreText) и поддержка вертикальной вёрстки; качество рендеринга в браузерах/программах отличается (отмечались ошибки в связке Chrome + Noto Sans Mongolian для Todo).

> Вывод: **Unicode‑ввод и рендер тодо‑бичиг возможны**, однако большая часть доступных онлайн‑данных — **сканы рукописей/ксилографов**. Это удобно для OCR, но для NMT/поисковых задач понадобятся параллели/транслитерации и лексикографическая опора.

## 1) Библиотеки, архивы и корпуса

### 1.1 Центр по развитию калмыцкого языка (ЦРКЯ) - раздел “Библиотека” (baylig.ru)
- **URL:** https://baylig.ru/library/
- **Что есть:** Категории *"Литература на Тодо бичиг"*, *"Рукописи на тодо бичиг"*, учебные и исторические материалы.
- **Формат:** PDF/сканы, иногда текст.
- **Unicode в тодо:** **скорее нет** (контент — изображения/сканы), но на сайте есть сопутствующие кириллические материалы (карточки слов и фраз, полезны для параллелей).


### 1.2 Научный архив Калмыцкого научного центра РАН (КалмНЦ РАН, Элиста)
- **URL:** https://kigiran.com/ - публикации об архивных фондах.
- **Что есть:** Буддийские и светские рукописи на тодо‑бичиг, фонды редких рукописей, описания единиц хранения, исследования с фрагментами и транслитерациями.
- **Формат:** Печатные каталоги, статьи (в формате PDF), иногда образцы сканов в публикациях.
- **Юридически:** Доступ/копирование по запросу архива, для публичного датасета нужна письменная договоренность.
- **Unicode в тодо:** **частично** в научных публикациях (транслитерации/транскрипции), но не как цельный Unicode‑корпус тодо.
- **Примечания:** Полезно для **эталонных примеров** нормализации/транслитерации, можно договариваться (наверное) об оцифровке и деперсонизации метаданных.


### 1.3 CNEAS, Университет Тохоку, Япония
- **URL (портал):** https://hkuri.cneas.tohoku.ac.jp/
- **Модули:** **K‑Search** (полнотекстовый поиск) и **K‑Dic** (многозадачные словари), включая **Todo**/**Oirat** разделы.
- **Что есть:** Полнотекстовые коллекции (Written Mongolian/Manchu/漢蒙), словари и справочники по **тодо‑бичиг/ойратскому**, есть **Integrated Dictionary of Written Oirat** (Kuribayashi, монография и онлайн-доступ).
- **Словарь:** **Integrated Dictionary of Written Oirat** (582 стр., монография + PDF).
- **Unicode в тодо:** **частично/да** — веб‑словарь и конвертер выдают Unicode‑последовательности тодо; PDF содержит примеры/таблицы.
- **Формат:** Веб‑интерфейс, иногда PDF; выгрузки не всегда доступны.
- **Юридически:** Академический доступ.
- **Примечания:** Отличный источник для **лексикографической валидации**, построения **глоссариев** и правил транслитерации.


### 1.4 Университет Кембриджа - материалы по "Tod Üsüg"
- **URL (пример):** https://www.repository.cam.ac.uk/items/109ddc08-1827-47bb-9174-03550bcbcb5c
- **Что есть:** Академические материалы о тодо‑бичиг, ссылки на источники.
- **Unicode в тодо:** **нет**
- **Количественно:** набор отдельных единиц (видео/посты)

## 4) Что уже известно про **Unicode‑версии** и **объёмы**

- На сегодня открытые **масштабные Unicode‑корпуса** тодо‑бичиг **не выявлены**; публично доступные массивы — это **сканы** (ACMS/DLIR **≈140 текстов**; региональные единичные коллекции; публикации КалмНЦ РАН — описательные).
- Для **лексикона/правил** доступны словари и конвертеры (Tohoku/CNEAS), что позволяет **генерировать/валидировать Unicode‑последовательности** тодо.
- В качестве параллелей для **перевода/NMT** доступны **корпуса калмыцкого (кириллица)** — НККЯ (~**800k** токенов)
